Directory Map

This being a map of the significant files and directories that come with your Moksa installation.

moksa/ - the directory root. This maps to wherever youve installed Moksa on your system.

moksa.jar - The main Moksa binary distribution. All the Moksa core class files are found here.

com/ - Moksa's Java source code. This is where all Java source code distributed with Moksa lives.

library/ - Moksa's Prolog source code. This is where all Prolog source code for Moksa's system libraries lives.

samples/ - Moksa program samples. In this directory you can find samples of various Moksa uses.

tests/ - Moksa test code.

doc/ - Moksa documentation.

Moksa can be found at http://www.svincent.com/moksa/
